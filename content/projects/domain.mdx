Designed and developed a Web3 domain naming system, utilizing blockchain technology to provide users with a decentralized and secure way to register and manage domain names. Leveraged Ethereum and IPFS technologies to store domain name information on the blockchain and ensure high availability and accessibility. Developed smart contracts and implemented various features, such as domain name registration, renewal, and transfer, using Solidity and other programming languages. Designed and developed the user interface for the domain naming system using React, HTML, and CSS, ensuring a seamless and user-friendly experience.
