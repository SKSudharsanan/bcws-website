Designed and developed a wallet for EVM chains, providing users with a secure and user-friendly platform to manage their digital assets. Utilized web3.js library to interact with Ethereum blockchain and other EVM-compatible chains. Implemented various features, such as private key management, token management, and transaction management, using Solidity and other programming languages. Designed and developed the user interface for the wallet using React, HTML, and CSS, ensuring a seamless user experience. Deployed the wallet on a decentralized hosting platform, ensuring high security and availability
